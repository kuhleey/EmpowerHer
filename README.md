EmpowerHer ZA

EmpowerHer ZA is a platform dedicated to empowering women in South Africa through education, mentorship, opportunities, and community-driven support.
Full website link :https://empowerherza.netlify.app/ 

 About the Project

This platform was created to:

Support women’s empowerment initiatives
Share educational and career resources
Connect women with mentors and opportunities
Build a safe and inspiring community space
Highlight stories, events, and programs that uplift women

The goal is to create an accessible digital space where women can learn, grow, and thrive.

 Features
Responsive modern UI
Inspirational landing page
Community-focused sections
